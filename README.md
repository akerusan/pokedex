# pokedex

・ポケモン一覧を表示する  
・無限スクロールさせ、offset を用いて 20 件ずつポケモンの一覧を表示する  
・項目をタップ時にポケモン詳細を表示する  
・オフライン時に過去に表示したポケモン一覧のキャッシュがある場合それを表示する  

利用した機能：

＊開発言語:Kotlin  
＊MVVM アーキテクチャ  
＊Android Architecture Component  
→ LiveData  
→ ViewModel  
→ Room  
→ DataBinding  
＊Retrofit  
＊Coroutine (DBにデータをプッシュするときのみ)  
＊ConstraintLayout
